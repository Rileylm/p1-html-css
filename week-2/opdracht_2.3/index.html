Waarom noem je de eerste HTML-pagina; index.html?

index.html is de naam die vaak wordt gebruikt voor de eerste HTML-pagina van een website. Webservers zoeken automatisch naar dit bestand als je een website opent zonder een specifieke pagina op te geven. Het zorgt ervoor dat de server weet welke pagina je als startpagina wilt tonen. Het is een standaard praktijk die al sinds de beginjaren van het web bestaat, omdat het makkelijk maakt om te bepalen welke pagina als eerste moet worden weergegeven. Het gebruik van index.html maakt websites overzichtelijk en voorkomt verwarring.


Met welke HTML versie werken we nu?

HTML5 (HyperText Markup Language 5) is de nieuwste, afgewerkte versie van de HTML-standaard.


Met welke CSS versie werken we nu?

CSS3 is de nieuwste versie van CSS, die nieuwe functies en verbeteringen introduceert in de stylingmogelijkheden van webontwikkeling. Het bouwt voort op de basis van CSS en biedt geavanceerde technieken voor het maken van visueel aantrekkelijke en responsieve ontwerpen.


Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De populariteit van webbrowsers varieert, maar wereldwijd is Google Chrome meestal de populairste browser. Volgens recente statistieken heeft Chrome doorgaans een marktaandeel van meer dan 77%, wat het ver boven andere browsers plaatst. Bij basisgebruik gebruikt google chrome 15% RAM en bij intesief gebruik 50% RAM


Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Dit komt omdat elke browser webcodes op zijn eigen manier interpreteert, wat leidt tot kleine verschillen in de weergave van je site.
